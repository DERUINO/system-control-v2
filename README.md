### Запуск (сервер)

#### В корневой директории выполнить команду

```
npm start
```

### Запуск (клиент)

#### В корневой директории выполнить команды (dev режим)

```
cd /front
npm run serve
```

#### В корневой директории выполнить команды (сборка клиента)

```
cd /front
npm run build
```
